How to run our program:
(a) Open config-db.js and add your own webuser password and port:
    password: '<YOUR PASSWORD>',
    port: '<YOUR PORT>',

(b) Install the following dependencies:
npm install mongodb express express-basic-auth path express-session nanoid@3.0.0

(c) Run your mongodb command in the terminal

(d) Run node main.js and you should now be able to listen on localhost:3000
